1. Apa yang dipahami mengenai Naming Convention
   Naming Convention merupakan cara untuk memberikan nama sebuah identifier seperti nama variabel, konstanta, method, dan class, agar sesuai dengan aturan yang berlaku pada Java dalam menulis sebuah kode. Pemberian nama pada sebuah identifier harus dilakukan secara teliti dan hati-hati untuk membuat program dapat dengan mudah dibaca dan dilakukan debug. Selain itu, naming convention diperlukan untuk mengurangi kesalahan dalam melakukan penulisan pada program.

2. Mengapa penamaan variabel dan class penting dalam pemrograman
   Penamaan variabel dan class penting dalam pemrograman dikarenakan program akan berjalan dengan cara memanggil variabel dan class tersebut untuk melakukan sebuah operasi. Variabel memungkinkan untuk menyimpan sebuah elemen yang dapat berubah-ubah. Maka dari itu, diperlukan penamaan yang tepat untuk membantu keterbacaan dari sebuah program dan mengetahui tujuan variabel dan class yang telah dibuat. Jika kode mudah dibaca, pemeliharaan kode akan lebih mudah dijalankan dan akan berguna jika ingin bekerja sama atau bertanya mengenai kode kepada orang lain.

4. Bagaimana operator aritmatika bekerja dalam Java
   Terdapat operator aritmatika dasar yang dapat digunakan dalam Java, di antaranya yaitu:
  a. Penjumlahan
  Digunakan untuk menjumlahkan nilai antara dua atau lebih variabel.
  Operator yang digunakan adalah = "+"
  b. Pengurangan
  Digunakan untuk mengurangkan nilai antara dua atau lebih variabel.
  Operator yang digunakan adalah = "-"
  c. Perkalian
  Digunakan untuk perkalian nilai antara dua atau lebih variabel.
  Operator yang digunakan adalah = "*"
  d. Pembagian
  Digunakan untuk membagi nilai antara dua atau lebih variabel.
  Operator yang digunakan adalah = "/"
  Pada operator pembagian, terkadang variabel yang digunakan akan menghasilkan angka desimal apabila dilakukan pembagian. Maka dari itu, diperlukan satu variabel dengan tipe data floating point seperti double atau menambahkan (double) sebelum melakukan output.
  e. Modulo
  Digunakan untuk mengetahui sisa bagi antar dua variabel.
  Operator yang digunakan adalah = "%"
  
  Selain itu, terdapat operator increment dan decrement yang dapat digunakan dalam Java untuk menambah atau mengurangi nilai variabel sebesar 1, di antaranya yaitu:
  a. Pre-increment: menambah x terlebih dahulu, lalu menggunakan nilainya (++x)
  b. Post-increment: menggunakan x terlebih dahulu, lalu menambah nilainya (++x)
  c. Pre-decrement: mengurangi x terlebih dahulu, lalu menggunakan nilainya (++x)
  d. Post-decrement: mengurangi x terlebih dahulu, lalu menggunakan nilainya (++x)
  
  Jika ingin menggabungkan operasi aritmatika tetapi ingin kode lebih ringkas, penulisan kode dapat menggunakan compound assigment operators
